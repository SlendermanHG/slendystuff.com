QwertyLock Paper

Paper-and-pen one-time pad companion for TXT/MMS. Use QwertyLock Secure for normal AES encryption.

Security Rule

This is only extremely secure if the pad numbers are truly random, kept secret, used once, and crossed off forever. Reusing pad numbers breaks the system.

The encrypted `QLP` block is SMS-safe visible text. The pad sheet is the secret key and must be shared separately before use, not in the same text thread.

Manual Alphabet

Write spaces as `_`. Supported paper text: A-Z, 0-9, space, period, comma, question mark.

By Hand

Encrypt

  1. Uppercase the message. Change spaces to `_`.
  2. Find each plain character number in the alphabet table.
  3. Take the next unused pad number.
  4. Add them. If the result is 40 or more, subtract 40.
  5. Write the result as two digits after `QLP`.

Decrypt

  1. Read each two-digit QLP number.
  2. Take the same pad numbers in the same order.
  3. Subtract the pad number. If negative, add 40.
  4. Look up the result in the alphabet table.
  5. Change `_` back to a space.

Make Local Pad Sheet

Generated in this browser with `crypto.getRandomValues`. Print or write down two matching copies. Do not send the pad with the encrypted message.
