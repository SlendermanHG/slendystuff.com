QwertyLock Ring

Paper-and-pen decrypter ring companion for TXT/MMS. Use QwertyLock Secure for normal AES encryption.

Security Rule

The ring is only secure when each key/pad symbol is random, secret, used once, and crossed off forever. Reusing a key stream breaks the paper system.

This page outputs QLR blocks. The ring replaces number math: align the current key symbol, then read the matching symbol on the ring.

Outer black ring is plaintext. Inner green ring is the rotating cipher ring. Touch or drag the ring to set the key symbol under outer A.

Ring Alphabet

Use `_` for a space. Supported symbols: A-Z, 0-9, underscore, period, comma, question mark.

By Hand

Encrypt

  1. Uppercase the message. Change spaces to `_`.
  2. Take the next unused key symbol.
  3. Rotate the inner ring so that key symbol sits under outer A.
  4. Find the plaintext symbol on the outer ring.
  5. Write the inner-ring symbol below it into the QLR block.

Decrypt

  1. Take the same key symbols in the same order.
  2. Rotate the inner ring so the key symbol sits under outer A.
  3. Find the encrypted symbol on the inner ring.
  4. Read the plaintext symbol from the outer ring.
  5. Change `_` back to a space.

Make Local Ring Key

Generated in this browser with crypto.getRandomValues. Print or write down two matching copies. Never send the key with the encrypted TXT/MMS message.
