QwertyLock Secure

SlendyStuff hidden-page encryption with TXT/MMS-safe output. The password is the decoder key.

Use 4+ random words or 20+ mixed characters.
Encrypted blocks are plain ASCII for TXT/MMS copy-paste.

Encrypted format: new messages start with QL2, then grouped QWERTY-safe text. No cover output, no hidden characters.

TXT/MMS: send the encrypted block as text, not as a screenshot. If the phone splits it into multiple SMS messages, the recipient needs the complete block in order.

Security: AES-GCM with a 256-bit key, random 32-byte salt, random 12-byte IV, and PBKDF2-SHA256. Old QL1 blocks still decrypt.

Paper mode: use QwertyLock Paper when you need a paper-and-pen one-time pad version. It uses QLP blocks, not QL2.

Decode rule: paste the encrypted block, enter the same password, press Decrypt. Without the password, the block is not recoverable.
